# Google Sheets для аналитика данных

Мини-курс по Google Sheets на примерах учета расходов, платежей, справочников, траков и операционной отчетности.

## Цель курса

После прохождения курса нужно уметь:

- понимать логику формул;
- писать основные формулы самостоятельно;
- строить рабочие таблицы без постоянного копирования чужих решений;
- проектировать небольшие системы учета;
- подготовиться к переходу на SQL и Power BI.

## Главный принцип

Не нужно запоминать сотни функций. Нужно сначала определить тип задачи.

| Что нужно сделать | Функция |
|---|---|
| Проверить условие | `IF` |
| Найти данные | `XLOOKUP` |
| Сложить по условию | `SUMIF` / `SUMIFS` |
| Посчитать по условию | `COUNTIF` / `COUNTIFS` |
| Оставить только нужные строки | `FILTER` |
| Отсортировать | `SORT` |
| Убрать дубликаты | `UNIQUE` |
| Подтянуть данные из другой книги | `IMPORTRANGE` |
| Применить одну формулу к столбцу | `ARRAYFORMULA` |
| Сделать сложную выборку | `QUERY` |

# 1. Ссылки

## Относительная ссылка

```gs
=A2
```

При копировании вниз станет `=A3`.

## Абсолютная ссылка

```gs
=$A$2
```

Не меняется при копировании.

## Полуабсолютные ссылки

```gs
=A$2
```

или

```gs
=$A2
```

---

# 2. IF

Схема:

```text
ЕСЛИ условие выполняется
ТО вернуть одно значение
ИНАЧЕ вернуть другое
```

Пример:

```gs
=IF(D2="Fuel","Yes","No")
```

Синтаксис:

```gs
=IF(условие,если_да,если_нет)
```

---

# 3. XLOOKUP

Используется, когда нужно найти значение в справочнике и вернуть связанное с ним значение.

Схема:

```gs
=XLOOKUP(
  что_ищем,
  где_ищем,
  что_вернуть,
  если_не_нашли
)
```

Пример:

```gs
=XLOOKUP(
  B2,
  Dictionary!A:A,
  Dictionary!C:C,
  "Unknown"
)
```

Читается так: найти компанию из `B2` в столбце `A` листа `Dictionary` и вернуть категорию из столбца `C`.

---

# 4. SUMIF

Используется, когда нужно сложить только те значения, которые соответствуют условию.

```gs
=SUMIF(
  A:A,
  "Fuel",
  B:B
)
```

Синтаксис:

```gs
=SUMIF(диапазон_условия,условие,диапазон_суммы)
```

---

# 5. COUNTIF

Используется, когда нужно посчитать количество строк, соответствующих условию.

```gs
=COUNTIF(D:D,"Unknown")
```

---

# 6. FILTER

Показывает только строки, которые соответствуют условию.

```gs
=FILTER(
  A:E,
  D:D="Unknown"
)
```

---

# 7. SORT

Сортирует диапазон.

```gs
=SORT(
  A:E,
  3,
  TRUE
)
```

Здесь `3` — номер столбца внутри выбранного диапазона, `TRUE` — сортировка по возрастанию.

---

# 8. UNIQUE

Возвращает список без повторений.

```gs
=UNIQUE(A:A)
```

---

# 9. IMPORTRANGE

Подтягивает данные из другой книги.

```gs
=IMPORTRANGE(
  "spreadsheet_id",
  "Dictionary!A:D"
)
```

Лучше использовать один раз на отдельном листе, например `Dictionary`, а все остальные формулы направлять уже в этот локальный диапазон.

Архитектура:

```text
Внешний справочник
        ↓
Локальный лист Dictionary
        ↓
XLOOKUP на рабочих листах
```

---

# 10. ARRAYFORMULA

Позволяет применить одну формулу сразу ко всему столбцу.

Идея:

```text
1000 строк → не 1000 формул, а одна
```

Пример:

```gs
=ARRAYFORMULA(IF(A2:A="","",B2:B*C2:C))
```

---

# 11. QUERY

Позволяет выбирать, фильтровать, сортировать и агрегировать данные в одной формуле.

Пример:

```gs
=QUERY(
  A:E,
  "select A, sum(E) where D = 'Fuel' group by A",
  1
)
```

`QUERY` — хороший мост к SQL.

# Частые ошибки

## Неверный диапазон

Нельзя:

```gs
A
```

Нужно:

```gs
A:A
```

## Неверный разделитель

В английской локали используются запятые:

```gs
=IF(A2>100,"Big","Small")
```

## IMPORTRANGE без диапазона

Нельзя:

```text
List!
```

Нужно:

```text
List!A:D
```

## Результат массива не может развернуться

Если `XLOOKUP` возвращает несколько столбцов, соседние ячейки должны быть пустыми.

## Не закреплена ссылка

Когда значение не должно двигаться при копировании, используй `$`.
